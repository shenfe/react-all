# react-all

模板项目，用于快速开始使用React及React官方或推荐库开发建站。

## 架构特性

单页面应用，前端渲染页面组件、控制路由。使用react、react-router、redux，结合webpack。

## 构建和启动命令

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm start

# build for production
npm run build

# run tests
npm run test
```
